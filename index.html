About FocusMC

FocusMC is a feature-rich Minecraft SMP server that supports both Java and Bedrock players (version 1.21+). Explore, survive, and thrive in a lag-free experience with amazing community features like events, ranks, rewards, PvP, economy, and auction systems.

We host weekly events, giveaways, and have a fully functional economy and rank system. Join us and become a part of the ultimate Minecraft family!
